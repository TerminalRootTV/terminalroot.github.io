### Dica rápida
{% highlight bash %}
vi /etc/vim/vimrc
{% endhighlight %}

### Com o arquivo aberto, corra com o cursor até abaixo da linha set ruler, pressione "i" sem as aspas e digite as linhas
{% highlight bash %}
set nu
syntax on 
{% endhighlight %}
