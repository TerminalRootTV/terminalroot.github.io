> Para esse feito utilizamos novamente o comando __dd__, com alguns parâmetros adicionais.

## Comando utilizado
{% highlight bash %}
dd if=/dev/cdrom of=/home/$USER/imagem.iso bs=2048 status=progress
{% endhighlight %}
